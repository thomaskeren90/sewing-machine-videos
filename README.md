# 🧵 Sewing Machine Videos — 90 Videos Project

## 📁 Project Structure

### 01-market-research/
Product data, pricing, content calendar, and descriptions.
- `PRODUCTS.md` — Full product catalog with prices & buy links
- `CALENDAR.md` — 90-video content calendar (30 days)
- `DESCRIPTIONS.md` — Ready-to-copy video descriptions & hashtags
- `product_catalog.xlsx` — Spreadsheet with all products, URLs, and prices

### 02-video-prompt-trial/
Yesterday's video production work — script templates, audio generation, and pipeline code.
- `scripts/` — 15 video type templates (01-unboxing through 15-closing-cta)
- `audio/` — 90 generated voiceover files (V001.mp3 – V090.mp3)
- `avatars/` — Avatar reference images
- `AI-VIDEO-WORKFLOW.md` — Complete production workflow guide
- `*.py` — Pipeline scripts (Replicate, Wan, parsing, generation)

### 03-video-trial-fail-101/
Today's video generation attempt — Day 1 output.
- `day1/audio/` — Generated Indonesian voiceover WAV files
- `day1/output/` — Generated MP4 videos (9:16 vertical, TikTok/Shorts ready)

## 🚀 Quick Start
1. Check `01-market-research/CALENDAR.md` for today's video schedule
2. Use scripts from `02-video-prompt-trial/scripts/` as templates
3. Generate videos following `02-video-prompt-trial/AI-VIDEO-WORKFLOW.md`
